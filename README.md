# COPILOT-CHALLENGE

PROJETO DE CONSTRUÇÃO DO COPILOT AGENT

DESCRIÇÃO: Este projeto apresenta um agente de IA especializado em astrologia e cartomancia, capaz de elaborar mapas astrais personalizados com base em dados fornecidos pelo cliente (nome completo, data, hora e local de nascimento). Com linguagem acessível e abordagem respeitosa, o agente interpreta posições planetárias, signos e aspectos astrológicos, conectando-os a temas práticos da vida como identidade, emoções, carreira, relacionamentos, saúde e espiritualidade. Opcionalmente, oferece leituras de Tarô como complemento simbólico e reflexivo.

OBJETIVO: Oferecer interpretações astrológicas confiáveis e práticas do mapa natal, traduzindo símbolos celestes em orientações aplicáveis ao cotidiano. Quando solicitado, complementa os insights com leituras de Tarô, promovendo autoconhecimento, reflexão e conexão espiritual, sempre com respeito às crenças do público e aos limites éticos da atuação esotérica.
